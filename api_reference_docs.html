POST /api/chat Chatbot con Groq AI — endpoint principal ▶
Body (JSON)
mensajerequiredstringPregunta en lenguaje natural sobre el negocio
id_comerciooptionalstringID del comercio a consultar. Si se omite, el bot pide que lo especifiques
Ejemplos de curl
# Producto más vendido
curl -X POST http://localhost:8000/api/chat \
  -H "Content-Type: application/json" \
  -d '{"mensaje": "¿Cuál fue mi producto más vendido?", "id_comercio": "COM001"}'
# Ticket promedio
curl -X POST http://localhost:8000/api/chat \
  -H "Content-Type: application/json" \
  -d '{"mensaje": "¿Cuál es mi ticket promedio?", "id_comercio": "COM001"}'
# Método de pago
curl -X POST http://localhost:8000/api/chat \
  -H "Content-Type: application/json" \
  -d '{"mensaje": "¿Qué método de pago usan más mis clientes?", "id_comercio": "COM002"}'
# Tendencia mensual
curl -X POST http://localhost:8000/api/chat \
  -H "Content-Type: application/json" \
  -d '{"mensaje": "¿Cómo han evolucionado mis ventas mes a mes?", "id_comercio": "COM003"}'
# Hora pico
curl -X POST http://localhost:8000/api/chat \
  -H "Content-Type: application/json" \
  -d '{"mensaje": "¿A qué hora del día vendo más?", "id_comercio": "COM003"}'
# Perfil de clientes
curl -X POST http://localhost:8000/api/chat \
  -H "Content-Type: application/json" \
  -d '{"mensaje": "¿Quiénes son mis mejores clientes?", "id_comercio": "COM001"}'
# Sin comercio — devuelve instrucción
curl -X POST http://localhost:8000/api/chat \
  -H "Content-Type: application/json" \
  -d '{"mensaje": "¿Cuánto vendí este mes?"}'
